# rankbot
Simple discord bot to track records for cEDH ranked matches.

Requirements:
|Package|Version|
|---|---|
|aiohttp|1.0.5|
|async-timeout|2.0.1|
|chardet|3.0.4|
|discord|0.0.2|
|discord.py|0.16.12|
|hashids|1.2.0|
|multidict|4.1.0|
|pip|9.0.1|
|pymongo|3.6.1|
|PyYAML|3.12|
|setuptools|38.6.0|
|websockets|3.4|
|wheel|0.30.0|

This bot uses MongoDB to store data.
A configuration file in config/bot.yaml stores the bot token and MongoDB connection information.
